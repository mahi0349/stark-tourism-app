# Tourism Management System

This is a complete Frontend implementation of the Tourism Management System, built according to the provided Software Requirement Specification (`tofan[2].docx`).

## Features Implemented Based on the Document:
- **Authentication & Roles:** Login and Registration pages simulating Tourist, Agent, and Admin roles (Reference: `User Table`).
- **Tour Packages:** A package listing with search and detailed cards (Reference: `Tour Package Table`).
- **Booking Flow:** The UI to browse and "book" tours, integrated within the packages and dashboard (Reference: `Booking Table`, `Payment Table`).
- **Admin/Agent Dashboard:** A mocked dashboard to manage Bookings, Tour Packages, Users, Payments, and Feedback. (Reference: `Admin Features` and tables).
- **Aesthetics & UI:** Ultra-premium glassmorphic dark theme built purely with CSS.

## Tech Stack
- React 18
- Vite
- React Router DOM
- Lucide React (for scalable icons)
- Vanilla CSS (`index.css`)

## How to Run
1. Open terminal and navigate to the `tourism-app` folder:
   ```bash
   cd "d:\Workspace\tf pro\tourism-app"
   ```
2. Install dependencies (already executed):
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the displayed local server URL (usually `http://localhost:5173/`) in your browser to view the application in action.
